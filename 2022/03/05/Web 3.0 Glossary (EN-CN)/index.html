Web 3.0 Glossary (EN-CN)

Web 3.0/DeFi/区块链等相关领域词汇的个人向记录, 仅供参考.

  • 本表无特定收录和排列规则.

  • "<>"里的中文表达为self coined.

English Full Form Mandarin
NFT Non-Fungible Token 非同质化通证 / 不可替代代币 / 数字藏品
AMM Automated Market Maker 自动化做市商
vAMM virtual AMM 虚拟自动化做市商
DAO Decentralized Autonomous Organization 去中心化自治组织
yield farming 收益耕作 / 流动性挖矿
DeFi Decentralized Finance 去中心化金融
CeFi Centralized Finance 集中式金融
DApp Decentralized Application 去中心化应用
DEX Decentralized Exchange 去中心化交易所
TVL Total Value Locked 总锁仓量
Binance 币安
BSC Binance Smart Chain 币安智能链
Terra 意为地球, 公链名
Avalanche 意为雪崩, 公链名
Solana 西班牙语中意为日光, 公链名
Polygon 意为多边形, 公链名
Fantom 意为幻影, 公链名
Bitcoin 比特币
altcoin alternative coin 替代币 / <比替币>
Litecoin 莱特币
USDC USD Coin 美元稳定币
USDT Tether USD 泰达币
UST TerraUSD <泰拉币>
airdrop 空投
Hardhat 意为安全帽, 主要基于TS/Solidity编写的Ethereum程序开发环境
Truffle 意为松露, 主要基于JS/TS/Solidity编写的以太坊程序开发环境
Curve 意为曲面, 以太坊上基于AMM的稳定币兑换协议
Cosmos 意为宇宙, 一种支持跨链交互的区块链生态
Polkadot 意为Polka圆点, 通常被称为波卡, 一个可跨链的区块链项目
oracle 预言机
GameFi Game Finance 链游 / 游戏化金融
meme coin 模因币 / <梗币>
IDO Initial DEX Offering <DEX通证首发>
ICO Initial Coin Offering 首次代币发行
DID Decentralized Identifiers 去中心化身份标识符
ERC Ethereum Request for Comment 以太坊意见征求
EIP Ethereum Improvement Proposal 以太坊改进提案
on-chain governance 链上治理
off-chain governance 链下治理
